# Cube Runner 3D

Prototype arcade 3D minimaliste où un cube avance automatiquement sur une piste suspendue.

## Gameplay

- Déplacement latéral sur 3 couloirs avec `←` et `→`.
- Saut avec `Espace` ou `↑`.
- Obstacles générés dynamiquement : murs, piques, trous.
- Pièces à collecter pour faire grimper le score.
- Accélération progressive pour augmenter la difficulté au fil de la distance.

## Lancer le jeu

Aucune dépendance externe :

```bash
python3 -m http.server 4173
```

Puis ouvrir `http://localhost:4173`.
